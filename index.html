Alchemyst's Arrows

El objetivo es conectar elementos desde las fuentes para crear reacciones y eliminar a todos los enemigos.

📜 Reglas

Elementos

  • Agua (5) - Conduce electricidad y apaga el fuego.
  • Rayo (7) - Se propaga por el agua y elimina enemigos mojados.
  • Grasa (11) - Propaga el fuego.
  • Fuego (13) - Quema grasa y elimina enemigos engrasados.

Enemigos

  • Enemigo (3)
  • Enemigo Mojado (17) - Vulnerable al rayo.
  • Enemigo Engrasado (29) - Vulnerable al fuego.
  • Enemigo Arruinado (31) - No puede ser eliminado.

Reacciones

  • Agua (5) + Rayo (7) → Electricidad en cadena.
  • Agua (5) + Fuego (13) → Ambos desaparecen.
  • Fuego (13) + Rayo (7) → Explosión.
  • Fuego (13) + Grasa (11) → Incendio en cadena.

Fuentes

  • Fuente A (19) - Puede conectarse a un único elemento.
  • Fuente B (23) - Puede conectarse a un único elemento.
  • Seleccioná un elemento y luego hacé clic en una fuente para asignarlo.
  • Una vez asignado, el elemento no puede cambiarse.
  • Los elementos se expanden desde las fuentes y sus conexiones.
